# React Job Listing Cards

A simple React project built with Vite that displays job listing cards posted by companies.
Each card shows the company name, job role, and posting information using reusable React components.

## Features

* Reusable React components
* Dynamic rendering using JavaScript arrays
* Clean card-based UI
* Built using Vite for fast development

## Tech Stack

* React
* Vite
* JavaScript
* CSS

## Project Structure

```
src/
 ├── components
 │    └── Card.jsx
 ├── App.jsx
 ├── App.css
 └── main.jsx
```

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/react-job-listing-ui.git
```

2. Install dependencies

```
npm install
```

3. Start the development server

```
npm run dev
```

This project was built to practice React concepts like components, props, and dynamic rendering.
